ESign → SideStore Importer

Paste an ESign source[…] share blob, decode it client-side, and get one-tap sidestore:// / altstore:// links for each source URL.

1. Decode an ESign source[…] blob

Format: ESign exports its source list as source[base64(bytes)], where the bytes are XOR-encrypted with a 5055-byte fixed key embedded in the ESign app. Algorithm from Backdoor/ESignRepoParser.swift; same key as ESignRepoKey.swift.

2. Source URLs

One URL per line. Tap Add to SideStore on your iPhone/iPad to import each.

    3. (Optional) Convert an ESign-style apps JSON → AltStore format

    Port of altSourceConverter. Fetches a plaintext ESign-style .json repo and groups it into AltStore-compatible JSON (apps with versions[]). Remote must be CORS-friendly, otherwise paste the JSON directly.
